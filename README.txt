# MindPath Website

A simple, beginner-friendly website about mental health support for young people in the UK.

## Files and Folders

- **index.html** - Main frameset page (start here)
- **header.html** - Top frame with website title
- **navigation.html** - Left frame with menu links
- **pages/** - Folder containing all 12 content pages
- **css/style.css** - Simple CSS styling

## 12 Pages

1. home.html - Introduction
2. mental-health.html - What is mental health
3. challenges.html - Common challenges (anxiety, depression, stress, burnout)
4. self-help.html - Self-help tips with ordered lists
5. getting-help.html - How to get help with links to NHS, Mind, YoungMinds
6. helplines.html - List of UK helplines
7. crisis.html - Crisis support information
8. parents.html - Guidance for parents and educators
9. get-involved.html - Ways to get involved
10. faq.html - Frequently asked questions
11. about.html - About the website
12. contact.html - Contact form

## How to Open the Website

### Best Method - Use Local Web Server

Open Terminal and run:

```
cd /Users/abdi/Library/CloudStorage/OneDrive-NewhamCollegeofFurtherEducation/MindPath-Website
python3 -m http.server 8000
```

Then open your browser and go to: **http://localhost:8000**

### Alternative - Direct Opening

Double-click **index.html** to open it directly in your browser.
